Roadmap

The Smart Screenshots project is continuously evolving. Here’s what’s in the pipeline:

  • Other Format Support: Expanding support to include additional file formats such as PDFs, videos, and more.
  • Larger Image Size Support: Improving the tool’s ability to handle larger image sizes for more accurate renaming.
  • More Cloud Integrations: Adding support for more cloud storage providers beyond Cloudinary.
  • Better NLP Models: Upgrading the underlying NLP models for even more accurate file renaming.
  • User Interface Improvements: Enhancing the user interface for easier navigation and better user experience.
